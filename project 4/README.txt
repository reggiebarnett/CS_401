INTRODUCTION

This program implements the Game of Life as described in the April
1970 issue of Scientific American.  The game is implemented in Java.
It is played on a 2-dimensional matrix of booleans and displayed in a
JFrame.  Animation is done by simply repainting the game, then
pausing, at each generation.

FILES

The following files should be present:

     GameofLife.java    - Source code.
    PaintPanel.java     - JFrame to display game.

USAGE

Compile in Java 2 SDK 1.5 with:

     javac GameofLife.java

Run with:

     java GameofLife

KNOWN PROBLEMS

None
